# The Derstand Programming Language

[![Ask DeepWiki](https://deepwiki.com/badge.svg)](https://deepwiki.com/Kuai-Ying-Studio/derstand)

## Why Derstand? (Because is not unDerstand)

- Written in Rust.
- Lightweight, fast, and ready to use.
- It is a superset of [Brainfuck](https://en.wikipedia.org/wiki/Brainfuck), extending it with more practical features.

## Installation

- Clone the repository:
  ```
  git clone https://github.com/Kuai-Ying-Studio/derstand.git
  ```
- Navigate to the project directory:
  ```
  cd derstand
  ```
- Build the project:
  ```
  cargo build --release
  ```
- The executable will be located in `target/release/derstand`.
